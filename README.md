# N7 Banking Website Clone

## Overview

This project is a responsive frontend implementation of the N7 Banking website based on the provided Figma design. The application is built using React.js and follows a component-based architecture to ensure maintainability, scalability, and code reusability.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Reusable React components
* Modern and clean user interface
* Smooth navigation and user interactions
* Component-based architecture
* Optimized and maintainable code structure

## Tech Stack

* React.js
* Vite
* JavaScript (ES6+)
* HTML5
* CSS3

## Installation

### Clone the Repository

```bash
git clone <repository-url>
```

### Navigate to the Project Directory

```bash
cd n7-banking-react
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
